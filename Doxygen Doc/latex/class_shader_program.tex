\doxysubsection{Detailed Description}
Manages a pair of vertex and pixel shaders and their input layout. 

Compiles HLSL shaders from a file, creates the corresponding shader objects (ID3\+D11\+Vertex\+Shader, ID3\+D11\+Pixel\+Shader), and sets up the \doxylink{class_input_layout}{Input\+Layout} based on the vertex shader\textquotesingle{}s input signature.
