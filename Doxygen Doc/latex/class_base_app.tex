\doxysubsection{Detailed Description}
Serves as the main application class, orchestrating the engine components. 

Initializes all necessary DirectX components (\doxylink{class_window}{Window}, \doxylink{class_device}{Device}, \doxylink{class_swap_chain}{Swap\+Chain}, etc.), manages the main application loop (message processing, update, render), and handles resource cleanup.
