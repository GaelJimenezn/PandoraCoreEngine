\doxysubsection{Detailed Description}
Manages the front and back buffers for presentation. 

Encapsulates an IDXGISwap\+Chain, handling the chain of buffers used to display rendered images on the screen. It also manages MSAA (Multisample Anti-\/\+Aliasing) settings.
