\doxysubsection{Detailed Description}
Clase de utilidad estática para cargar modelos 3D desde archivos .obj. 

Proporciona métodos para parsear la geometría (vértices, texturas, normales) y las caras de un archivo .obj, optimizando los datos para su renderizado.
