\doxysubsection{Detailed Description}
Manages a Direct3D 11 Render Target View (RTV). 

This class wraps an ID3\+D11\+Render\+Target\+View, which binds a resource (typically a texture) to the output-\/merger stage to serve as a rendering destination.
