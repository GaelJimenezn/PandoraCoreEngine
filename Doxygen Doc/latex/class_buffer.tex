\doxysubsection{Detailed Description}
Manages a generic Direct3D 11 buffer resource (Vertex, Index, Constant). 

Encapsulates an ID3\+D11\+Buffer, handling its creation, updating, binding to the pipeline, and destruction. Can represent different buffer types based on the bind flags provided during initialization.
