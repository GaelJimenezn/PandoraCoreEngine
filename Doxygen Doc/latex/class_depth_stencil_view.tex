\doxysubsection{Detailed Description}
Manages a Direct3D 11 Depth Stencil View (DSV). 

This class encapsulates an ID3\+D11\+Depth\+Stencil\+View, which allows a texture resource to be used as a depth and stencil buffer in the output-\/merger stage of the graphics pipeline.
