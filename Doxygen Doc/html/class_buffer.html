Pandora Core Engine Docs
Pandora Core Engine is a 3D rendering engine built from scratch in C++ and DirectX 11. This project encapsulates key DirectX components (Device, Context, SwapChain, Resources) into reusable classes and demonstrates a basic rendering pipeline capable of displaying a rotating, textured cube. Created as part of a 3D Computer Graphics course.

Detailed Description

Manages a generic Direct3D 11 buffer resource (Vertex, Index, Constant).

Encapsulates an ID3D11Buffer, handling its creation, updating, binding to the pipeline, and destruction. Can represent different buffer types based on the bind flags provided during initialization.

Binds the buffer to the appropriate pipeline stage.

Binds as Vertex Buffer (IASetVertexBuffers), Index Buffer (IASetIndexBuffer), or Constant Buffer (VSSetConstantBuffers and optionally PSSetConstantBuffers).

Parameters
deviceContextThe device context for command submission.
StartSlotThe starting slot index for binding.
NumBuffersThe number of buffers to bind (usually 1).
setPixelShaderIf true and it's a Constant Buffer, also binds to the Pixel Shader stage.
formatRequired for Index Buffers (e.g., DXGI_FORMAT_R32_UINT). Ignored otherwise.
