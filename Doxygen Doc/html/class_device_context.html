Pandora Core Engine Docs
Pandora Core Engine is a 3D rendering engine built from scratch in C++ and DirectX 11. This project encapsulates key DirectX components (Device, Context, SwapChain, Resources) into reusable classes and demonstrates a basic rendering pipeline capable of displaying a rotating, textured cube. Created as part of a 3D Computer Graphics course.

Detailed Description

Wraps the Direct3D 11 device context for command submission.

This class encapsulates the ID3D11DeviceContext, which is responsible for generating rendering commands, binding resources to the graphics pipeline, and executing draw calls.

ID3D11DeviceContext* DeviceContext::m_deviceContext = nullptr

Pointer to the immediate context of Direct3D 11.

This interface is used to issue all rendering commands. It is acquired during initialization and released upon destruction.
